# Verbs Overview

Verbs represent actions and operations that can be performed on nouns within your data model. This section covers how to define, manage, and utilize verbs in your AI applications.

## Key Features

- Define action types and their properties
- Create relationships between verbs and nouns
- Organize and categorize operations
- Integrate verbs with AI functions and workflows

## Getting Started

To start working with verbs, define your action types in the admin interface and use them in your AI applications to create a structured operational model.
