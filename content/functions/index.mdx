# [Functions.do](https://functions.do) Composable Units of Work

> **Predictable AI results through elegant, type-safe functions**

## Setup

Install the `functions.do` package:

```sh npm2yarn
npm i functions.do
```

Then [get your API key](https://apis.do/console) and set it in an environment variable:

```env .env
DO_API_KEY=sk_do_*****************************
```
