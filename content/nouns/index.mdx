# Nouns Overview

Nouns represent the core entities and concepts in your data model. This section covers how to define, manage, and utilize nouns within your AI applications.

## Key Features

- Define entity types and their properties
- Create relationships between different nouns
- Organize and categorize your data model
- Integrate nouns with AI functions and workflows

## Getting Started

To start working with nouns, define your entity types in the admin interface and use them in your AI applications to create a structured data model.
