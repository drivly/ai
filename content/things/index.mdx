# Things Overview

Things represent the core resources and entities in your data model. This section covers how to define, manage, and utilize things within your AI applications.

## Key Features

- Define resource types and their properties
- Create relationships between different things
- Organize and categorize your resources
- Integrate things with AI functions and workflows

## Getting Started

To start working with things, define your resource types in the admin interface and use them in your AI applications to create a structured data model.
