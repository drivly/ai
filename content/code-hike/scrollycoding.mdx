# ScrollyCoding Example

This page demonstrates the ScrollyCoding feature of CodeHike with Nextra v4.

<CH.Scrollycoding>

```js
function hello() {
  return 'Hello'
}
```

Step 1: We start with a simple function that returns "Hello".

```js focus=2
function hello() {
  return 'Hello'
}
```

Step 2: Let's focus on the return statement.

```js focus=2
function hello() {
  return 'Hello, World!'
}
```

Step 3: Now we've changed the return value to "Hello, World!".

```js focus=1:3
function hello() {
  console.log('Greeting someone')
  return 'Hello, World!'
}
```

Step 4: We've added a console.log statement to log what we're doing.

```js focus=1:4
function hello(name) {
  console.log(`Greeting ${name}`)
  return `Hello, ${name}!`
}
```

Step 5: Finally, we've modified the function to accept a name parameter and use it in both the log and return statements.

</CH.Scrollycoding>
