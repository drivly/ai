# Searches Overview

Searches enable you to query and retrieve data within your AI applications. This section covers how to define, manage, and execute searches as part of your workflows and agents.

## Key Features

- Define reusable search operations
- Query across multiple data sources
- Integrate with vector databases
- Filter and sort search results

## Getting Started

To start working with searches, define your query operations in the admin interface and incorporate them into your workflows and agents.
