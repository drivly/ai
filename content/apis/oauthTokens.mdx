# OauthTokens API

API for managing OauthTokens resources.

## Fields

### token

- **Type**: `text`
- **Required**: Yes




### provider

- **Type**: `text`
- **Required**: Yes




### userId

- **Type**: `relationship`
- **Required**: Yes




### clientId

- **Type**: `text`
- **Required**: Yes




### expiresAt

- **Type**: `date`
- **Required**: Yes




### scope

- **Type**: `text`
- **Required**: No





## API Endpoints

### GET /api/oauthTokens
Retrieves a list of OauthTokens with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/oauthTokens/:id
Retrieves a single OauthTokens by ID.

### POST /api/oauthTokens
Creates a new OauthTokens.

### PATCH /api/oauthTokens/:id
Updates an existing OauthTokens.

### DELETE /api/oauthTokens/:id
Deletes a OauthTokens.
