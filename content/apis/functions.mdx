# Functions API

API for managing Functions resources.

## Fields

### name

- **Type**: `text`
- **Required**: Yes




### type

- **Type**: `select`
- **Required**: No
- **Default**: `Generation`



### public

- **Type**: `checkbox`
- **Required**: No
- **Default**: `false`
- **Description**: Make this function available to other users


### clonedFrom

- **Type**: `relationship`
- **Required**: No

- **Description**: Original function this was cloned from


### pricing

- **Type**: `group`
- **Required**: No

- **Description**: Monetization settings for this function


### format

- **Type**: `select`
- **Required**: No
- **Default**: `Object`



### schemaYaml

- **Type**: `code`
- **Required**: No




### shape

- **Type**: `json`
- **Required**: No




### code

- **Type**: `code`
- **Required**: No




### prompt

- **Type**: `relationship`
- **Required**: No




### role

- **Type**: `text`
- **Required**: No




### user

- **Type**: `relationship`
- **Required**: No




### agent

- **Type**: `relationship`
- **Required**: No




### examples

- **Type**: `relationship`
- **Required**: No

- **Description**: Example arguments for this function



## API Endpoints

### GET /api/functions
Retrieves a list of Functions with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/functions/:id
Retrieves a single Functions by ID.

### POST /api/functions
Creates a new Functions.

### PATCH /api/functions/:id
Updates an existing Functions.

### DELETE /api/functions/:id
Deletes a Functions.
