# Errors API

API for managing Errors resources.

## Fields

### message

- **Type**: `text`
- **Required**: Yes

### stack

- **Type**: `textarea`
- **Required**: No

- **Description**: Error stack trace

### digest

- **Type**: `text`
- **Required**: No

- **Description**: Error digest for identifying specific errors

### url

- **Type**: `text`
- **Required**: No

- **Description**: URL where the error occurred

### source

- **Type**: `text`
- **Required**: No

- **Description**: Source of the error (client/server/etc)

## API Endpoints

### GET /api/errors

Retrieves a list of Errors with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/errors/:id

Retrieves a single Errors by ID.

### POST /api/errors

Creates a new Errors.

### PATCH /api/errors/:id

Updates an existing Errors.

### DELETE /api/errors/:id

Deletes a Errors.
