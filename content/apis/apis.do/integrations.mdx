## API Endpoints

### GET /api/integrations
Retrieves a list of Integrations with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/integrations/:id
Retrieves a single Integrations by ID.

### POST /api/integrations
Creates a new Integrations.

### PATCH /api/integrations/:id
Updates an existing Integrations.

### DELETE /api/integrations/:id
Deletes a Integrations.
