# OauthClients API

API for managing OauthClients resources.

## Fields

### name

- **Type**: `text`
- **Required**: Yes




### clientId

- **Type**: `text`
- **Required**: Yes




### clientSecret

- **Type**: `text`
- **Required**: Yes




### redirectURLs

- **Type**: `array`
- **Required**: Yes




### disabled

- **Type**: `checkbox`
- **Required**: No
- **Default**: `false`



### createdBy

- **Type**: `relationship`
- **Required**: No





## API Endpoints

### GET /api/oauthClients
Retrieves a list of OauthClients with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/oauthClients/:id
Retrieves a single OauthClients by ID.

### POST /api/oauthClients
Creates a new OauthClients.

### PATCH /api/oauthClients/:id
Updates an existing OauthClients.

### DELETE /api/oauthClients/:id
Deletes a OauthClients.
