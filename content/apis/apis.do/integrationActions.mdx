## API Endpoints

### GET /api/integrationActions
Retrieves a list of IntegrationActions with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/integrationActions/:id
Retrieves a single IntegrationActions by ID.

### POST /api/integrationActions
Creates a new IntegrationActions.

### PATCH /api/integrationActions/:id
Updates an existing IntegrationActions.

### DELETE /api/integrationActions/:id
Deletes a IntegrationActions.
