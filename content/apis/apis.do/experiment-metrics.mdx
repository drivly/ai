Metrics collected from real-world user feedback for experiments

## API Endpoints

### GET /api/experiment-metrics
Retrieves a list of Experiment-metrics with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/experiment-metrics/:id
Retrieves a single Experiment-metrics by ID.

### POST /api/experiment-metrics
Creates a new Experiment-metrics.

### PATCH /api/experiment-metrics/:id
Updates an existing Experiment-metrics.

### DELETE /api/experiment-metrics/:id
Deletes a Experiment-metrics.
