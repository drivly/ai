## API Endpoints

### GET /api/apikeys
Retrieves a list of Apikeys with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/apikeys/:id
Retrieves a single Apikeys by ID.

### POST /api/apikeys
Creates a new Apikeys.

### PATCH /api/apikeys/:id
Updates an existing Apikeys.

### DELETE /api/apikeys/:id
Deletes a Apikeys.
