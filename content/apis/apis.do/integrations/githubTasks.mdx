# GithubTasks API

Tasks triggered by GitHub events




## Fields

```typescript
interface GithubTasks {
  issueNumber: number;
  repository: string;
  jobId: string;
  status: 'processing' | 'completed' | 'failed';
}
```




## API Endpoints

### GET /api/githubTasks
Retrieves a list of GithubTasks with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/githubTasks/:id
Retrieves a single GithubTasks by ID.

### POST /api/githubTasks
Creates a new GithubTasks.

### PATCH /api/githubTasks/:id
Updates an existing GithubTasks.

### DELETE /api/githubTasks/:id
Deletes a GithubTasks.
