## API Endpoints

- `GET /api/roles` - List all Roles
- `GET /api/roles/:id` - Get a Role by ID
- `POST /api/roles` - Create a new Role
- `PATCH /api/roles/:id` - Update a Role
- `DELETE /api/roles/:id` - Delete a Role
