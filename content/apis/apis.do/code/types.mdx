# Types API

Defines TypeScript type definitions used throughout the system




## Fields

```typescript
interface Types {
  name?: string;
  hash?: string;
  type?: string;
  json?: Record<string, any>;
  schema?: Record<string, any>;
}
```




## API Endpoints

### GET /api/types
Retrieves a list of Types with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/types/:id
Retrieves a single Types by ID.

### POST /api/types
Creates a new Types.

### PATCH /api/types/:id
Updates an existing Types.

### DELETE /api/types/:id
Deletes a Types.
