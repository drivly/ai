# Deployments API

Manages deployments of code to production environments




## Fields

```typescript
interface Deployments {
  name?: string;
}
```




## API Endpoints

### GET /api/deployments
Retrieves a list of Deployments with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/deployments/:id
Retrieves a single Deployments by ID.

### POST /api/deployments
Creates a new Deployments.

### PATCH /api/deployments/:id
Updates an existing Deployments.

### DELETE /api/deployments/:id
Deletes a Deployments.
