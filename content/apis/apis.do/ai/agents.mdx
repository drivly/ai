# Agents API

Autonomous AI agents that can perform tasks using functions and workflows




## Fields

```typescript
interface Agents {
  name?: string;
  public?: boolean;
  clonedFrom?: string;
  pricing?: any;
  goals?: string[];
}
```




## API Endpoints

### GET /api/agents
Retrieves a list of Agents with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/agents/:id
Retrieves a single Agents by ID.

### POST /api/agents
Creates a new Agents.

### PATCH /api/agents/:id
Updates an existing Agents.

### DELETE /api/agents/:id
Deletes a Agents.
