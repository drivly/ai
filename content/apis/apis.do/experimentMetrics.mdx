## API Endpoints

### GET /api/experimentMetrics
Retrieves a list of ExperimentMetrics with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/experimentMetrics/:id
Retrieves a single ExperimentMetrics by ID.

### POST /api/experimentMetrics
Creates a new ExperimentMetrics.

### PATCH /api/experimentMetrics/:id
Updates an existing ExperimentMetrics.

### DELETE /api/experimentMetrics/:id
Deletes a ExperimentMetrics.
