# Nouns API

Defines semantic noun entities with their various grammatical forms




## Fields

```typescript
interface Nouns {
  name?: string;
  singular?: string;
  plural?: string;
  possessive?: string;
  pluralPossessive?: string;
  verb?: string;
  act?: string;
  activity?: string;
  event?: string;
  order?: number;
  group?: string;
  type?: Array<{ relationTo: string; value: string }>;
  resources?: any;
}
```




## API Endpoints

### GET /api/nouns
Retrieves a list of Nouns with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/nouns/:id
Retrieves a single Nouns by ID.

### POST /api/nouns
Creates a new Nouns.

### PATCH /api/nouns/:id
Updates an existing Nouns.

### DELETE /api/nouns/:id
Deletes a Nouns.
