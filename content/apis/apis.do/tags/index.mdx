## API Endpoints

- `GET /api/tags` - List all Tags
- `GET /api/tags/:id` - Get a Tag by ID
- `POST /api/tags` - Create a new Tag
- `PATCH /api/tags/:id` - Update a Tag
- `DELETE /api/tags/:id` - Delete a Tag
