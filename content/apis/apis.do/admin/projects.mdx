# Projects API

Manages projects and their associated domains




## Fields

```typescript
interface Projects {
  name?: string;
  domain?: string;
  domains?: string[];
  goals?: string[];
}
```




## API Endpoints

### GET /api/projects
Retrieves a list of Projects with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/projects/:id
Retrieves a single Projects by ID.

### POST /api/projects
Creates a new Projects.

### PATCH /api/projects/:id
Updates an existing Projects.

### DELETE /api/projects/:id
Deletes a Projects.
