# Config API

Configuration for .ai folder synchronization




## Fields

```typescript
interface Config {
  path: string;
  contentHash: string;
  project: string;
  lastSyncedAt: string;
  lastModifiedLocally?: string;
  lastModifiedRemotely?: string;
  lastModifiedGithub?: string;
  repository?: string;
  syncStatus: 'synced' | 'conflict' | 'pending';
  data?: Record<string, any>;
  syncConfig?: any;
}
```




## API Endpoints

### GET /api/config
Retrieves a list of Config with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/config/:id
Retrieves a single Config by ID.

### POST /api/config
Creates a new Config.

### PATCH /api/config/:id
Updates an existing Config.

### DELETE /api/config/:id
Deletes a Config.
