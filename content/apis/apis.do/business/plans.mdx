# Plans API

Strategic plans with actionable steps to achieve goals




## Fields

```typescript
interface Plans {
  name: string;
  description?: string;
  status?: 'draft' | 'active' | 'completed' | 'archived';
  startDate?: string;
  endDate?: string;
  owner?: string;
  goals?: string[];
  tags?: string[];
  priority?: 'low' | 'medium' | 'high' | 'critical';
  steps?: any[];
}
```




## API Endpoints

### GET /api/plans
Retrieves a list of Plans with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/plans/:id
Retrieves a single Plans by ID.

### POST /api/plans
Creates a new Plans.

### PATCH /api/plans/:id
Updates an existing Plans.

### DELETE /api/plans/:id
Deletes a Plans.
