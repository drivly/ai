## API Endpoints

### GET /api/integrationCategories
Retrieves a list of IntegrationCategories with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/integrationCategories/:id
Retrieves a single IntegrationCategories by ID.

### POST /api/integrationCategories
Creates a new IntegrationCategories.

### PATCH /api/integrationCategories/:id
Updates an existing IntegrationCategories.

### DELETE /api/integrationCategories/:id
Deletes a IntegrationCategories.
