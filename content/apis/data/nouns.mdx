# Nouns API

API for managing Nouns resources.

## Fields

### name

- **Type**: `text`
- **Required**: No




### singular

- **Type**: `text`
- **Required**: No

- **Description**: Singular form like User


### plural

- **Type**: `text`
- **Required**: No

- **Description**: Plural form like Users


### possessive

- **Type**: `text`
- **Required**: No

- **Description**: Possessive form like User's


### pluralPossessive

- **Type**: `text`
- **Required**: No

- **Description**: Plural possessive form like Users'


### verb

- **Type**: `text`
- **Required**: No

- **Description**: Related verb like Use


### act

- **Type**: `text`
- **Required**: No

- **Description**: Third person singular present tense like Uses


### activity

- **Type**: `text`
- **Required**: No

- **Description**: Gerund like Using


### event

- **Type**: `text`
- **Required**: No

- **Description**: Past tense like Used


### resources

- **Type**: `join`
- **Required**: No





## API Endpoints

### GET /api/nouns
Retrieves a list of Nouns with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/nouns/:id
Retrieves a single Nouns by ID.

### POST /api/nouns
Creates a new Nouns.

### PATCH /api/nouns/:id
Updates an existing Nouns.

### DELETE /api/nouns/:id
Deletes a Nouns.
