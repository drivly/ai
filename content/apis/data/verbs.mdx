# Verbs API

API for managing Verbs resources.

## Fields

### action

- **Type**: `text`
- **Required**: No

- **Description**: Active tense like Create

### act

- **Type**: `text`
- **Required**: No

- **Description**: Third person singular present tense like Creates

### activity

- **Type**: `text`
- **Required**: No

- **Description**: Gerund like Creating

### event

- **Type**: `text`
- **Required**: No

- **Description**: Past tense like Created

### subject

- **Type**: `text`
- **Required**: No

- **Description**: Subject like Creator

### object

- **Type**: `text`
- **Required**: No

- **Description**: Object like Creation

### inverse

- **Type**: `text`
- **Required**: No

- **Description**: Opposite like Destroy

### inverseAct

- **Type**: `text`
- **Required**: No

- **Description**: Third person singular present tense like Destroys

### inverseActivity

- **Type**: `text`
- **Required**: No

- **Description**: Gerund like Destroying

### inverseEvent

- **Type**: `text`
- **Required**: No

- **Description**: Past tense like Destroyed

### inverseSubject

- **Type**: `text`
- **Required**: No

- **Description**: Subject like Destroyer

### inverseObject

- **Type**: `text`
- **Required**: No

- **Description**: Object like Destruction

## API Endpoints

### GET /api/verbs

Retrieves a list of Verbs with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/verbs/:id

Retrieves a single Verbs by ID.

### POST /api/verbs

Creates a new Verbs.

### PATCH /api/verbs/:id

Updates an existing Verbs.

### DELETE /api/verbs/:id

Deletes a Verbs.
