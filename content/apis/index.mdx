# APIs.do

## Clickable Developer Experiences

APIs.do provides a unified gateway for accessing all services in the AI Primitives platform. It enables developers to create, deploy, and manage APIs with a simple, declarative syntax.

## Features

- **Unified API Gateway**: Access all services through a single, consistent interface
- **Clickable Links**: Create interactive API endpoints that can be accessed via URLs
- **Type Safety**: Full TypeScript support for reliable development
- **Authentication & Authorization**: Built-in security features
- **Rate Limiting**: Protect your APIs from abuse
- **Observability**: Monitor API usage and performance
- **Documentation**: Automatically generate API documentation

## Usage

```typescript
import { API } from 'apis.do'

// Define an API endpoint
export default API(async (request, context) => {
  // Extract parameters from the request
  const { name } = request.query

  // Process the request
  const greeting = `Hello, ${name || 'World'}!`

  // Return a response
  return {
    api: {
      name: 'greeting-api',
      description: 'A simple greeting API',
      url: 'https://apis.do/greeting',
      version: '1.0.0',
    },
    greeting,
    timestamp: new Date().toISOString(),
  }
})
```
