# API Reference

> **Comprehensive documentation for all platform APIs**

## Overview

The API Reference provides detailed documentation for all RESTful endpoints across the platform. This centralized resource enables developers to understand and integrate with the various services and capabilities offered by the .do ecosystem.

## Core API Categories

### AI Primitives

- **Functions API** - Manage and execute strongly-typed AI functions
- **Workflows API** - Orchestrate business processes and integrations
- **Agents API** - Control autonomous digital workers

### Data Management

- **Database API** - Access and manipulate structured data
- **Things API** - Work with the universal data model
- **Nouns API** - Manage entity types and classifications
- **Verbs API** - Define actions and relationships

### Event System

- **Triggers API** - Configure event-driven process initiation
- **Searches API** - Retrieve contextual data
- **Actions API** - Execute operations on external systems

### Observability

- **Analytics API** - Access usage and performance metrics
- **Generations API** - Track AI generation history
- **Logs API** - Retrieve system and application logs
