## API Endpoints

### GET /api/integrationTriggers

Retrieves a list of IntegrationTriggers with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/integrationTriggers/:id

Retrieves a single IntegrationTriggers by ID.

### POST /api/integrationTriggers

Creates a new IntegrationTriggers.

### PATCH /api/integrationTriggers/:id

Updates an existing IntegrationTriggers.

### DELETE /api/integrationTriggers/:id

Deletes a IntegrationTriggers.
