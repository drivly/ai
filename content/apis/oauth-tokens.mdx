# Oauth-tokens API

API for managing Oauth-tokens resources.

## Fields

### token

- **Type**: `text`
- **Required**: Yes




### provider

- **Type**: `text`
- **Required**: Yes




### userId

- **Type**: `relationship`
- **Required**: Yes




### clientId

- **Type**: `text`
- **Required**: Yes




### expiresAt

- **Type**: `date`
- **Required**: Yes




### scope

- **Type**: `text`
- **Required**: No





## API Endpoints

### GET /api/oauth-tokens
Retrieves a list of Oauth-tokens with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/oauth-tokens/:id
Retrieves a single Oauth-tokens by ID.

### POST /api/oauth-tokens
Creates a new Oauth-tokens.

### PATCH /api/oauth-tokens/:id
Updates an existing Oauth-tokens.

### DELETE /api/oauth-tokens/:id
Deletes a Oauth-tokens.
