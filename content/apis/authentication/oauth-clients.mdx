# Oauth-clients API

API for managing Oauth-clients resources.

## Fields

### name

- **Type**: `text`
- **Required**: Yes




### clientId

- **Type**: `text`
- **Required**: Yes




### clientSecret

- **Type**: `text`
- **Required**: Yes




### redirectURLs

- **Type**: `array`
- **Required**: Yes




### disabled

- **Type**: `checkbox`
- **Required**: No
- **Default**: `false`



### createdBy

- **Type**: `relationship`
- **Required**: No





## API Endpoints

### GET /api/oauth-clients
Retrieves a list of Oauth-clients with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/oauth-clients/:id
Retrieves a single Oauth-clients by ID.

### POST /api/oauth-clients
Creates a new Oauth-clients.

### PATCH /api/oauth-clients/:id
Updates an existing Oauth-clients.

### DELETE /api/oauth-clients/:id
Deletes a Oauth-clients.
