# EvalResults API

API for managing EvalResults resources.

## Fields

### name

- **Type**: `text`
- **Required**: Yes

### testId

- **Type**: `relationship`
- **Required**: Yes

- **Description**: Reference to the evaluation test this result is for

### output

- **Type**: `json`
- **Required**: No

- **Description**: Output data from running the test

### score

- **Type**: `number`
- **Required**: No

- **Description**: Overall score for this evaluation result (0-1)

### metrics

- **Type**: `json`
- **Required**: No

- **Description**: Detailed metrics for this evaluation result

### duration

- **Type**: `number`
- **Required**: No

- **Description**: Duration of the test execution in milliseconds

### error

- **Type**: `text`
- **Required**: No

- **Description**: Error message if the test execution failed

## API Endpoints

### GET /api/evalResults

Retrieves a list of EvalResults with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/evalResults/:id

Retrieves a single EvalResults by ID.

### POST /api/evalResults

Creates a new EvalResults.

### PATCH /api/evalResults/:id

Updates an existing EvalResults.

### DELETE /api/evalResults/:id

Deletes a EvalResults.
