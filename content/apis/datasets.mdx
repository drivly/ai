# Datasets API

API for managing Datasets resources.

## Fields

### name

- **Type**: `text`
- **Required**: No





## API Endpoints

### GET /api/datasets
Retrieves a list of Datasets with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/datasets/:id
Retrieves a single Datasets by ID.

### POST /api/datasets
Creates a new Datasets.

### PATCH /api/datasets/:id
Updates an existing Datasets.

### DELETE /api/datasets/:id
Deletes a Datasets.
