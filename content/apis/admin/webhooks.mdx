# Webhooks API

API for managing Webhooks resources.

## Fields

### name

- **Type**: `text`
- **Required**: No




### url

- **Type**: `text`
- **Required**: Yes




### filters

- **Type**: `array`
- **Required**: No




### enabled

- **Type**: `checkbox`
- **Required**: No
- **Default**: `true`



### secret

- **Type**: `text`
- **Required**: No





## API Endpoints

### GET /api/webhooks
Retrieves a list of Webhooks with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/webhooks/:id
Retrieves a single Webhooks by ID.

### POST /api/webhooks
Creates a new Webhooks.

### PATCH /api/webhooks/:id
Updates an existing Webhooks.

### DELETE /api/webhooks/:id
Deletes a Webhooks.
