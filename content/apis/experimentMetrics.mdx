# ExperimentMetrics API

Metrics collected from real-world user feedback for experiments

## Fields

### experimentId

- **Type**: `relationship`
- **Required**: Yes

- **Description**: The experiment this metric is associated with


### variantId

- **Type**: `text`
- **Required**: Yes

- **Description**: The variant ID this metric is for


### userId

- **Type**: `text`
- **Required**: No

- **Description**: User ID associated with this metric (if available)


### sessionId

- **Type**: `text`
- **Required**: No

- **Description**: Session ID associated with this metric (if available)


### metricName

- **Type**: `text`
- **Required**: Yes

- **Description**: Name of the metric (e.g., click_through_rate, conversion_rate)


### value

- **Type**: `number`
- **Required**: Yes

- **Description**: Numeric value of the metric


### timestamp

- **Type**: `date`
- **Required**: No

- **Description**: When this metric was recorded


### metadata

- **Type**: `json`
- **Required**: No

- **Description**: Additional metadata for this metric (e.g., browser, device, page)



## API Endpoints

### GET /api/experimentMetrics
Retrieves a list of ExperimentMetrics with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/experimentMetrics/:id
Retrieves a single ExperimentMetrics by ID.

### POST /api/experimentMetrics
Creates a new ExperimentMetrics.

### PATCH /api/experimentMetrics/:id
Updates an existing ExperimentMetrics.

### DELETE /api/experimentMetrics/:id
Deletes a ExperimentMetrics.
