# Roles API

API for managing Roles resources.

## Fields

### name

- **Type**: `text`
- **Required**: Yes




### superAdmin

- **Type**: `checkbox`
- **Required**: No
- **Default**: `false`
- **Description**: Grant super admin privileges to users with this role



## API Endpoints

### GET /api/roles
Retrieves a list of Roles with pagination.

#### Query Parameters
- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/roles/:id
Retrieves a single Roles by ID.

### POST /api/roles
Creates a new Roles.

### PATCH /api/roles/:id
Updates an existing Roles.

### DELETE /api/roles/:id
Deletes a Roles.
