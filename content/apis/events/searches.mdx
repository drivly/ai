# Searches API

API for managing Searches resources.

## Fields

### name

- **Type**: `text`
- **Required**: No

### query

- **Type**: `text`
- **Required**: No

### searchType

- **Type**: `select`
- **Required**: No
- **Default**: `text`

### results

- **Type**: `json`
- **Required**: No

### embedding

- **Type**: `json`
- **Required**: No

## API Endpoints

### GET /api/searches

Retrieves a list of Searches with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/searches/:id

Retrieves a single Searches by ID.

### POST /api/searches

Creates a new Searches.

### PATCH /api/searches/:id

Updates an existing Searches.

### DELETE /api/searches/:id

Deletes a Searches.
