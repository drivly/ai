# Triggers API

API for managing Triggers resources.

## Fields

### name

- **Type**: `text`
- **Required**: No

### payload

- **Type**: `json`
- **Required**: No

### config

- **Type**: `json`
- **Required**: No

## API Endpoints

### GET /api/triggers

Retrieves a list of Triggers with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/triggers/:id

Retrieves a single Triggers by ID.

### POST /api/triggers

Creates a new Triggers.

### PATCH /api/triggers/:id

Updates an existing Triggers.

### DELETE /api/triggers/:id

Deletes a Triggers.
