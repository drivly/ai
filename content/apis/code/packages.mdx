# Packages API

API for managing Packages resources.

## Fields

### name

- **Type**: `text`
- **Required**: No

### package

- **Type**: `json`
- **Required**: No

- **Description**: The package.json content for publishing to NPM

### collections

- **Type**: `array`
- **Required**: No

- **Description**: Collections to include in this package

## API Endpoints

### GET /api/packages

Retrieves a list of Packages with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/packages/:id

Retrieves a single Packages by ID.

### POST /api/packages

Creates a new Packages.

### PATCH /api/packages/:id

Updates an existing Packages.

### DELETE /api/packages/:id

Deletes a Packages.
