# Types API

API for managing Types resources.

## Fields

### name

- **Type**: `text`
- **Required**: No

### hash

- **Type**: `text`
- **Required**: No

### type

- **Type**: `code`
- **Required**: No

### json

- **Type**: `json`
- **Required**: No

### schema

- **Type**: `json`
- **Required**: No

## API Endpoints

### GET /api/types

Retrieves a list of Types with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/types/:id

Retrieves a single Types by ID.

### POST /api/types

Creates a new Types.

### PATCH /api/types/:id

Updates an existing Types.

### DELETE /api/types/:id

Deletes a Types.
