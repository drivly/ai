# Evals API

API for managing Evals resources.

## Fields

### name

- **Type**: `text`
- **Required**: Yes

### description

- **Type**: `textarea`
- **Required**: No

### input

- **Type**: `json`
- **Required**: No

- **Description**: Input data for the evaluation test

### expected

- **Type**: `json`
- **Required**: No

- **Description**: Expected output data for comparison

### tags

- **Type**: `array`
- **Required**: No

- **Description**: Tags for categorizing and filtering tests

## API Endpoints

### GET /api/evals

Retrieves a list of Evals with pagination.

#### Query Parameters

- `limit`: Maximum number of items to return (default: 10)
- `page`: Page number for pagination (default: 1)
- `sort`: Field to sort by (default: createdAt)
- `where`: JSON query for filtering results

### GET /api/evals/:id

Retrieves a single Evals by ID.

### POST /api/evals

Creates a new Evals.

### PATCH /api/evals/:id

Updates an existing Evals.

### DELETE /api/evals/:id

Deletes a Evals.
