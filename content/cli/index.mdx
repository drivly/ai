# CLI

The CLI can be installed globally via `npm i -g apis.do`.

```bash
npm i -g cli.do
```

or can use `npx` to run commands:

```bash
npx functions.do deploy
```

## `functions.do`

```bash
functions.do <command>
```

### `init`

```bash
functions.do init
```

### `login`

```bash
functions.do login
```
