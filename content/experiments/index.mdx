# Experiments Overview

The Experiments collection provides tools and services for experimenting with AI models and configurations.

## Components

- **Experiments**: Run and track experiments
- **Models**: Manage AI models
- **Prompts**: Design and optimize prompts
- **Settings**: Configure experiment parameters

To get started with the Experiments collection, explore the following pages:
