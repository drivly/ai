# Resources Overview

Resources represent the core entities and objects in your data model. This section covers how to define, manage, and utilize resources within your AI applications.

## Key Features

- Define resource types and their properties
- Create relationships between different resources
- Organize and categorize your data model
- Integrate resources with AI functions and workflows

## Getting Started

To start working with resources, define your resource types in the admin interface and use them in your AI applications to create a structured data model.
