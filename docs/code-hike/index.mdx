# CodeHike Examples

This page demonstrates the integration of CodeHike with Nextra v4.

## Basic Code Block

```js
function hello() {
  console.log("Hello, CodeHike!");
}
```

## Annotated Code

```js
function hello() {
  // This is a simple function
  console.log("Hello, CodeHike!"); // This will print to the console
}
```

## Code with Focus

```js focus=2
function hello() {
  console.log("Hello, CodeHike!");
}
```

## Code with Line Numbers

```js showLineNumbers
function hello() {
  console.log("Hello, CodeHike!");
}
```

## Code with Line Highlighting

```js {2}
function hello() {
  console.log("Hello, CodeHike!");
}
```

## Code with Multiple Line Highlighting

```js {1,3}
function hello() {
  console.log("Hello, CodeHike!");
  return "Hello";
}
```

## Code with Range Highlighting

```js {1-3}
function hello() {
  console.log("Hello, CodeHike!");
  return "Hello";
}
```

## Code with Line Highlighting and Focus

```js {2} focus=2
function hello() {
  console.log("Hello, CodeHike!");
}
```
